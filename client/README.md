# Frontend (React + Vite)

## Setup

1. Create environment file:

```bash
cp .env.example .env
```

2. Install dependencies:

```bash
npm install
```

3. Start frontend:

```bash
npm run dev
```

App runs on `http://localhost:5173`.

## Features

- Landing page
- Signup/Login with OTP and Google auth
- Dashboard
- Split-screen portfolio builder with Monaco editor and live preview
- Subscription plans with Razorpay checkout
- Contact page
- Admin analytics dashboard with Chart.js
